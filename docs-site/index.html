üöÄ Quick Start

Get Started with QuantDesk Protocol

Deploy and start trading on the decentralized protocol:

# 1. Clone and setup
git clone https://github.com/dextrorsal/quantdesk.git
cd quantdesk
npm install

# 2. Build smart contracts
cd contracts/smart-contracts
anchor build
anchor deploy --provider.cluster devnet

# 3. Start backend
cd ../../backend
npm install
npm run dev

# 4. Start frontend
cd ../frontend
npm install
npm run dev

# 5. Access QuantDesk
open http://localhost:5173
