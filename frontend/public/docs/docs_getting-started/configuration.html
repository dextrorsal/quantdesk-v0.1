üîë Exchange API Setup

Bitget (Recommended for Live Trading)

  1. Create Account: Sign up at bitget.com
  2. Enable API: Go to Account ‚Üí API Management
  3. Create API Key:
  4. Name: "QuantDesk Trading"
  5. Permissions: Enable "Spot Trading" and "Read"
  6. IP Restrictions: Add your IP address
  7. Get Credentials:
  8. API Key
  9. Secret Key
  10. Passphrase

Binance (For Market Data)

  1. Create Account: Sign up at binance.com
  2. Enable API: Go to Account ‚Üí API Management
  3. Create API Key:
  4. Name: "QuantDesk Data"
  5. Permissions: Enable "Read Info" only
  6. Get Credentials:
  7. API Key
  8. Secret Key

Coinbase (Alternative Data Source)

  1. Create Account: Sign up at coinbase.com
  2. Enable API: Go to Settings ‚Üí API
  3. Create API Key:
  4. Name: "QuantDesk Data"
  5. Permissions: Read-only
  6. Get Credentials:
  7. API Key
  8. Secret Key

üö® Common Configuration Issues

"API Key Invalid"

  • Check API key is correct
  • Verify API permissions
  • Ensure IP restrictions allow your IP

"Permission Denied"

  • Check API key has trading permissions
  • Verify account is verified
  • Check exchange account status

"Rate Limit Exceeded"

  • Reduce request frequency
  • Use multiple API keys
  • Implement request queuing

"Data Not Found"

  • Check data path exists
  • Verify exchange symbols
  • Run data fetching script
